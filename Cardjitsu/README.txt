# Juego de Cartas - Reglas y Manual del Jugador

## Descripción del Juego
El Juego de Cartas es un emocionante juego de cartas elemental en el que compites contra la computadora. Cada jugador tiene una mano de cartas con elementos y niveles de poder. El objetivo es ganar encuentros y obtener victorias sobre la computadora.

## Reglas del Juego
El juego se juega con las siguientes reglas:

1. Elementos del Juego:
   - Fuego vence a Nieve.
   - Nieve vence a Agua.
   - Agua vence a Fuego.

2. Niveles de Poder:
   - Cada carta tiene un nivel de poder de 1 a 10.

3. Empates:
   - En caso de empate entre los elementos, el desempate se decide por la carta con mayor nivel de poder.

4. Ganar una Partida:
   - Ganas la partida si obtienes 3 victorias de ventaja 
   - La partida también puede terminar en empate después de 7 encuentros.

## Instrucciones de Juego
Sigue estas instrucciones para jugar:

1. Ejecución:
   - Ejecuta el programa para iniciar el juego.

2. Reglas:
   - Al comienzo, se mostrarán las reglas del juego en la pantalla.

3. Marcador:
   - El marcador muestra la cantidad de encuentros, victorias del jugador, victorias de la computadora y empates.

4. Mano Inicial:
   - Cada jugador comienza con una mano de cartas.

5. Seleccionar una Carta:
   - Haz clic en una de tus cartas para seleccionarla.

6. Computadora Escoge:
   - La computadora seleccionará aleatoriamente una carta.

7. Resultado:
   - El juego determinará el ganador según las reglas.
   - Se mostrará el resultado en la pantalla.

8. Siguiente Ronda:
   - Se dibujarán las cartas seleccionadas en el tablero.
   - El marcador se actualizará.
   - Repite los pasos del 5 al 7 para cada encuentro.

9. Ganar la Partida:
   - Gana la partida si obtienes 3 victorias más que la computadora.
   - El juego mostrará un mensaje de victoria.

10. Empate:
    - Si después de 7 encuentros no hay un ganador claro, el juego termina en empate.

11. Finalizar el Juego:
    - El juego se cerrará automáticamente después de mostrar el resultado final.
